carsus.io.cmfgen.base.find_row(fname, string1, string2='', how='both', num_row=False)¶

Search strings inside plain text files and returns matching line or row number.

Parameters
fnamestr

Path to plain text file.

string1str

String to search.

string2str

Extra string to search (default is ‘’).

how{‘one’, ‘both’, ‘first’}

If ‘both’ search for string1 AND string2. If ‘one’ search for string1 OR string2. If ‘first’ searches for ‘string1’ AND NOT string2 (default is ‘both’).

num_rowbool

If true, returns row number instead (default is False).

Returns
str or int

Returns matching line or match row number.
